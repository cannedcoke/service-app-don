# Don 🇵🇾

> Conectá personas con proveedores verificados 

**Don** es una aplicación web estática que funciona como directorio de servicios profesionales en Paraguay. Permite a los usuarios encontrar electricistas, plomeros, diseñadores, personal de limpieza y más, contactándolos directamente sin intermediarios.

🔗 **Demo en vivo:** https://cannedcoke.github.io/service-app-don/

---

## Páginas

| Ruta | Descripción |
|---|---|
| `index.html` | Landing page principal |
| `views/recomendaciones.html` | Listado de proveedores |
| `views/providers.html` | Formulario para registrarse como proveedor |
| `views/detalle.html` | Detalle de un proveedor |
| `views/aboutUs.html` | Sobre nosotros, términos y contacto |

---

##  Cómo ejecutar localmente


1. Cloná el repositorio

```bash
git clone https://github.com/cannedcoke/service-app-don.git
cd service-app-don
```

2. Abrí `index.html` directamente en el navegador, o usá la extensión **Live Server** de VS Code para tener recarga automática.

---

## ☁️ Cómo desplegar en GitHub Pages

1. Subí el repositorio a GitHub
2. Andá a **Settings → Pages**
3. En *Source*, seleccioná la rama `main` y la carpeta `/ (root)`
4. Guardá — en unos segundos tenés la URL pública:
